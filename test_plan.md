# 🧪 GrowWithMe - 機能検証テスト計画 (単語リスト編)

## 1. プロフィール管理機能
**目的**: 個人データが正しく保存・維持されるかを確認する。

- [ ] **入力テスト**: `/profile` ページで以下の項目を入力してください。
    - 基本: 名前、年齢、性別
    - 背景: 職業、家族構成、出身地
    - 興味: 趣味 (例: "サッカー")
    - その他キーワード: (例: "Google, ラーメン")
    - トーン: "Spartan" (厳しめ) か "Friendly" (親しみ) に変更
- [ ] **保存確認**: 「Save Profile」を押し、ページをリロード (F5) して、入力内容が消えていないか確認。
- [ ] **日本語確認**: 日本語（例：「大阪」）が文字化けせずに残っているか確認。

## 2. 単語リスト管理機能
**目的**: 単語の追加、削除、一括登録ができるかを確認する。

- [ ] **個別追加**: "Single" タブから単語を1つ追加 (例: `apple`)。
- [ ] **一括登録**: "Bulk" タブに変え、 `banana, cherry, date` と入力して Import を押す。3単語追加されるか確認。
- [ ] **削除**: 単語を削除ボタンで消し、リストから消えるか確認。

## 3. データバックアップ (JSON)
**目的**: データのエクスポートと復元ができるかを確認する。

- [ ] **書き出し**: 「Export Backup」を押し、ファイル (`gwm_backup_xxxx.json`) を保存する。
- [ ] **読み込み**: 「Import Backup」で先ほどの JSON を読み込む。プロフィールと単語が元通りになるか確認。

## 4. AI例文生成 (単語学習モード)
**目的**: 単語学習に適した「簡潔な例文」が生成されるかを確認する。

- [ ] **生成実行**: 単語リスト画面で **"Generate Sentences"** ボタンを押す。
- [ ] **内容チェック**: 生成後、単語の行をクリックして詳細を開く。
    - **形式**: **English** (例文) と **JP** (訳) が表示されているか。
    - **長さ**: 1文の短い例文（5〜10単語程度）になっているか。
    - **関連性**: 例文に「職業」や「趣味」の要素が入っているか。
    - **コンテキスト**: AIの解説 (Context) に理由が書かれているか。
- [ ] **保存確認**: ページをリロードしても、生成された例文が消えずに残っているか。

## 5. 話題バランシング
**目的**: 話題が偏らず、適度に分散されるかを確認する。

- [ ] **1回目**: 3単語ほど例文生成し、使われた話題（例：「家族」）をメモする。
- [ ] **2回目**: 新しい単語を3つ追加して、再度生成する。
- [ ] **バランス確認**: 2回目の生成では、さっき使った「家族」以外の話題（例：「仕事」や「出身地」）が優先されているか確認する。
